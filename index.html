Distributed Acoustic Sensing (DAS) Simulator

This web app is not associated with any organization, and was not requested in any official way. At this time it is intended to be an experimental demonstration of the technology

DAS technology transforms existing fiber optic cable into a continuous sensing system by detecting vibration and acoustic signatures.

This web app is a simulator to create roadway events that are displayed on a waterfall plot.

Currently, only Big Cottonwood Canyon is on the map.
